# User Stories

## Event Creation and Management
- As a user, I want to create an event, so that I can invite others to participate in an activity.
- As a user, I want to edit the details of my event, so that I can keep the information updated.
- As a user, I want to cancel an event I created, so that participants are informed when the event can no longer take place.
- As a user, I want to view the list of registered attendees for my event, so that I can track participation.
- As a user, I want to define the maximum number of attendees for my event, so that I can manage available capacity.

## Event Discovery and Browsing
- As a user, I want to browse upcoming events, so that I can find activities that interest me.
- As a user, I want to filter events by category, so that I can quickly find relevant events.
- As a user, I want to view event details, so that I can know the schedule, venue, and available spots.
- As a user, I want to identify the status of an event, so that I know whether it is open, ongoing, completed, or cancelled.

## Registration and Waiting List
- As a user, I want to register for an available event, so that I can attend it.
- As a user, I want to join a waiting list when an event is full, so that I can get a spot if one becomes available.
- As a user, I want to cancel my registration, so that another participant may take my place.
- As a user, I want to receive confirmation when I register successfully, so that I know my spot is reserved.

## Reviews and Ratings
- As a user, I want to rate an event after attending it, so that I can share my experience with others.
- As a user, I want to leave a written review for a past event, so that future participants can make informed decisions.
- As a user, I want to read reviews from previous attendees, so that I can evaluate whether an event is worth joining.

## Administrative Actions
- As an administrator, I want to edit any event, so that I can correct inappropriate or inaccurate information.
- As an administrator, I want to remove events that violate platform rules, so that the platform remains safe and organized.
- As an administrator, I want to manage categories, so that events are properly classified.
- As an administrator, I want to manage venues, so that users can choose valid locations for their events.
- As an administrator, I want to manage user accounts, so that I can moderate the platform when necessary.
